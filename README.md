# Есть план — синяя версия

Второй визуальный вариант интерактивного мобильного навигатора по самостоятельной взрослой жизни. В этой версии используется синяя дизайн-система и персонаж-помощник Планчик.

## Локальный запуск

```bash
npm install
npm run dev
```

## Публикация на GitHub Pages

1. Создайте на GitHub новый публичный репозиторий.
2. Загрузите в него содержимое этой папки и отправьте ветку `main`.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите `Source: GitHub Actions`.
5. Дождитесь завершения задачи `Deploy to GitHub Pages` во вкладке `Actions`.

После каждого обновления ветки `main` сайт будет пересобираться автоматически.

## Демонстрационный сценарий

1. На главной открыть маршрут «Первая работа».
2. Ответить на три вопроса.
3. Открыть персональный план из шести шагов.
4. Отмечать шаги выполненными и показать сохранение прогресса.

Прогресс хранится локально в браузере. Для повторной демонстрации откройте
`Профиль` и нажмите `Сбросить демонстрацию`.
